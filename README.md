# SmartBudget

Landing page para app de finanzas personales.

## Tecnologías

- HTML5 semántico
- SASS (patrón 7-1)
- Metodología BEM
- Bootstrap 4
- Responsive mobile-first

## Instalación

Compila SASS:
```bash
sass sass/main.scss css/main.css --watch
```

Abre `index.html` en el navegador.
